## ScuClawer
基于Scrapy框架的川大网站爬虫
 
 ### **CirSpider - 教室信息发布系统爬虫**
 - 网页地址: http://202.115.47.164/cir/index.html
 - 获取数据: 当天所有教室占用信息
 - 格式: JSON
 - 示例: `ScuClawer/Product/CirSpider`
    - `classrooms.json`: 所有教室
    - `courses.json`: 所有课程,(此文件经过手动修改以适应JSON格式)
